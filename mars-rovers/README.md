NASA Mars Rovers — Student Project
----------------------------------

This website was created for educational purposes to showcase NASA’s Mars Rover missions.
Includes information about Curiosity, Perseverance, and the Mars Exploration Rovers.

Structure:
  /mars-rovers/
    index.html
    mission-history.html
    faq.html
    /css/styles.css

To publish:
1. Create a GitHub repo named mars-rovers.
2. Upload these files.
3. Enable GitHub Pages in Settings → Pages.
4. Visit the generated URL to view your live site.
